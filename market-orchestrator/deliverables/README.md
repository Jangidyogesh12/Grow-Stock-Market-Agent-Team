# Deliverables

Place outputs here that should survive beyond a single run.

Required structure:

- `daily-reports/` for one markdown market report per date
- `stock-tracking/` for per-stock CSV files of current investments

Expected upstream handoffs before final synthesis:

- `broker/deliverables/YYYY-MM-DD-broker-handoff.md`
- `company-finance-tracker/deliverables/YYYY-MM-DD-finance-handoff.md`
- `news-reporter/deliverables/YYYY-MM-DD-news-handoff.md`

Suggested dated report filename:

- `YYYY-MM-DD-market-report.md`
